# ヒミツ発見メーカー

ウェブページの画像にマウスを乗せると、レンズ越しに「ヒミツの画像」が覗けるブラウザ拡張機能です。  
Grok（xAI）の画像変換APIを使って、元画像から自動でヒミツ画像を生成します。

## 必要なもの

- Google Chrome または Microsoft Edge
- [xAI APIキー](https://console.x.ai/) または ※[VeniceAI APIキー](https://venice.ai/settings/api)（どちらの場合も、Grok imagine image APIを使います）

  (※外部サービス。月額+少し割高なAPI利用料がかかりますが、xAI公式のAPIよりモデレーション基準が緩い可能性があります)

---

## インストール方法

**Chrome の場合**

1. このページの緑の「**Code**」ボタン → 「**Download ZIP**」でダウンロード
2. ZIPを解凍する
3. `chrome://extensions` を開く
4. 右上の「**デベロッパーモード**」をオンにする
5. 「**パッケージ化されていない拡張機能を読み込む**」をクリック
6. 解凍したフォルダの中の **`v3`フォルダ**を選択する

**Edge の場合**

手順は Chrome とほぼ同じです。`edge://extensions` を開いて同様に操作してください。

---

## 初期設定

1. 拡張機能のアイコンを右クリック →「**オプション**」を開く
2. **API接続先 / APIキー**で、接続先を選択し、対応するAPIキーを入力（`xai-` で始まる文字列）
3. **プロンプトセット**を設定する（後述）
4. 「保存」ボタンを押す

### プロンプト設定例

プロンプトは「元画像をどう変換するか」の指示です。日本語でも英語でも書けます。

| セット名 | プロンプト例 |
|---|---|
| 水着 | `wearing a bikini, beach background` |
| 透け | `sheer transparent clothing, same pose` |
| ○○風 | （自分でアレンジしてください） |

## 使い方

### 画像を選ぶ（2つの方法）

**クリックモード（おすすめ）**  
拡張機能のアイコンをクリックして「クリックで選択」→ 変換したい画像の上でクリック

**ドラッグモード**  
「ドラッグで範囲指定」→ 変換したい部分をドラッグで囲む  
→ CORSエラーで取得できない画像もこのモードなら対応可能

### レンズゲーム

画像が生成されると自動でレンズゲームが始まります。

| 操作 | 動作 |
|---|---|
| マウスを動かす | レンズがついてくる |
| `Ctrl + ↑` / `Ctrl + ↓` | レンズサイズを変更 |
| アイコン再クリック | ゲーム終了 |

---

## 注意事項

- APIキーはあなたのブラウザ内（`chrome.storage.local`）にのみ保存されます。外部には送信しません
- grok imagine image APIの利用料は xAI または VeniceAI のアカウントに従います
- 生成されたヒミツ画像は設定したフォルダに自動保存できます（オプションで設定）
